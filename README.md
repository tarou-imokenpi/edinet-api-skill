# edinet-api skill

EDINET API（金融庁 電子開示システム）V2 のインターフェース仕様を提供する Agent Skillです。

## 構成

```
SKILL.md                    # スキル本体（早見表）
references/
  api-spec.md               # 詳細仕様リファレンス
```

## インストール

`SKILL.md` と `references/` を `~/.copilot/skills/edinet-api/` に配置する。
