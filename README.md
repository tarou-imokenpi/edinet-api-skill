# edinet-api skill

EDINET API（金融庁 電子開示システム）V2 のインターフェース仕様を提供する Copilot スキル。

## 構成

```
SKILL.md                    # スキル本体（早見表）
references/
  api-spec.md               # 詳細仕様リファレンス
```

## インストール

`SKILL.md` と `references/` を `~/.copilot/skills/edinet-api/` に配置する。

## ソース

[EDINET API仕様書（Version 2）](https://disclosure2dl.edinet-fsa.go.jp/guide/static/disclosure/WZEK0030.html) — 金融庁（2026年1月版）
